# Automotive Diagnostic Tools

Python-based tools for BS6 telematics configuration and UDS ECU flash validation.
Developed for commercial vehicle platforms (Ashok Leyland, Olectra EV Bus).
### UDS Flash Analyzer
Analyses CAN bus logs for UDS (ISO 14229) flash sequences.
Generates PDF RCA reports with latency analysis, issue detection, and session diagrams.

**Run:** `python uds_flash_analyzer.py`

## Requirements
pip install openpyxl reportlab

## Domain
Automotive Telematics | CAN Bus | UDS ISO 14229 | J1939 | BS6 | Python
